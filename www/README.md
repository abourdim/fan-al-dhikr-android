# 🤲 The Art of Dhikr & Dua — فن الذكر والدعاء

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **فن الذكر والدعاء عند خاتم الأنبياء (The Art of Dhikr and Dua)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 lesson cards on dhikr and dua with Quran verses and practical actions
- Daily adhkar section with morning/evening remembrances
- Daily habit tracker with streaks
- Knowledge quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes with localStorage persistence
- Narrator (audiobook mode with karaoke, voice/speed/pitch controls)
- Responsive design + PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks, no dependencies. Just open `index.html`.
